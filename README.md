# 🔐 CRYPT-PW Generator (AltDB / IRR)

A web-based password generator for CRYPT-PW (DES Crypt), compatible with IRR and AltDB.  
Simple, fast, and secure for generating authentication-ready hashes.

---

## ⚙️ Features

- 🔑 Generate CRYPT-PW hashes (DES Crypt)
- ⚡ Fast and lightweight
- 👁 Toggle password visibility
- 📋 Copy result with one click
- 🌐 100% browser-based (no backend required)

## 🚀 How to use

1. Enter your password  
2. Click **"GENERATE CRYPT-PW"**  
3. Copy the generated hash  
4. Paste it into your IRR / AltDB  

---

## ⚠️ Disclaimer

This project uses DES Crypt, which is considered **legacy and not secure for modern applications**.  
It is intended only for compatibility with systems that still require CRYPT-PW (e.g., IRR / AltDB).
